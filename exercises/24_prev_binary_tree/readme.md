# 04 二叉树的前序遍历

## 题目

实现二叉树的前序遍历（根 → 左 → 右），输出遍历结果。

## 要求

- 二叉树节点包含 `val`（整数值）、`left` 和 `right` 指针。
- 输入：通过层次遍历数组构建二叉树（如 `[1,2,3,null,null,4,5]` 表示根 1，左 2，右 3，3 的左 4，右 5）。
- 输出：前序遍历结果 `1 2 3 4 5`（示例）。

## 测试输入输出

- 输入层次数组 `[1,2,3,null,null,4,5]` → 输出：`1 2 3 4 5`。

## 考察点

- 递归/非递归遍历实现。
- 二叉树的构造（处理 null 节点）。
