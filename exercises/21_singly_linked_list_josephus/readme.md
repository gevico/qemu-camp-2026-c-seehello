# 01 单链表实现约瑟夫环问题

- 题目

  用单链表模拟约瑟夫环问题。给定总人数 `n`、起始报数位置 `k`（从 1 开始计数）、报数阈值 `m`（数到 `m` 的人出列），按出列顺序输出所有人的编号。

- 要求

  - 链表节点包含 `id`（编号）和 `next` 指针。
  - 输入：`n=5, k=1, m=2`（5 人围成环，从 1 号开始数，数到 2 出列）。
  - 输出：`2 → 4 → 1 → 5 → 3`（示例）。

- 测试输入输出

  - 输入：`n=5, k=1, m=2` → 输出：`2 4 1 5 3`（用空格分隔）。

- **考察点：链表遍历、循环指针操作、边界条件（如 `m = 1` 时直接全部出列）。**
